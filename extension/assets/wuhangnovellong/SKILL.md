# wuhangnovellong

## Usage
触发指令示例：`你好``加载武行小说技能包`、`我想写一本书`、`我想写一本小说`、`项目初始化`、`55555`

## Steps



0.  **欢迎与初始预期 (Welcome & Initial Expectation)**
    - **欢迎**：向用户打招呼：“你好我是武行，欢迎使用长篇小说创作技能包。”
    - **思考建议**：提示用户：“在正式流程开始之前，建议你花几分钟思考对本书的预期（题材边界、你预期的一些设定和人物剧情、个人执念等）。可以在脑海中过一遍，也可以简单记下来，放在 `1-边界/预期.md`文件里面。”

1.  **模式路由 (Mode Routing) - 启动阶段**
    -  向用户展示三种启动模式并等待选择：
        1. **🎨 标准模式（从零开始）**：读取 `define.md`，通过交互定义套路方向和文风。
        2. **📂 存量/灵活模式（带资进组）**：读取 `guide.md`，扫描并整合已有稿件，或从特定点（如先写人设）切入。
        3. **📚 参考模式（有样板书）**：读取 `analyze.md`，拆解对标作品的文风、套路和节奏。

2.  **创意与设定阶段 (Ideation Phase)**
    - 启动阶段完成后，进入创意孵化。
    - 读取 `ideation.md`，完成创意脑暴、新书设定案、金手指与角色细化，并将所有设定注册到 `SOLOENT.md` 索引区。

3.  **大纲规划阶段 (Outline Phase)**
    - 读取 `outline.md`，按需执行全书总纲、单卷卷纲或标准化章纲细化。
    - **灵活使用**：可以分批进行（如：写完第一卷章纲 → 写第一卷正文 → 再回来写第二卷卷纲）

4.  **正文写作阶段 (Writing Phase)**
    - 读取 `write.md`，执行单章创作循环（前置检查 → 撰写 → 校准 → 状态更新）。
    - **循环阻断**：每章完成后必须暂停，等待用户指令。

5.  **质检与迭代 (Review Phase)**
    - 读取 `review.md`，可在任意阶段插入（大纲质检、正文质检、设定回溯）。

6.  **状态追踪与导航 (Status Tracking)**
    - 用户输入 `检查进度` 或 `下一步做什么` 时：
        - 读取 `SOLOENT.md §8`（项目进度与待办）
        - 根据当前阶段推荐下一步操作
        - 提示可调用的子技能


### 🔗 子技能索引

- **Analyze（样板书分析）**: `analyze.md` - 深度拆解样板书，提取文风、套路、节奏。**注意**：步骤 3 套路拆解须**严格按 analyze.md 规定的九大分析维度与格式参考**生成 `1.3_套路方向.md`，不可自创结构或省略任一维度。
- **Define（方向定义）**: `define.md` - 无样板书时，通过交互定义文风和套路方向
- **Guide（灵活启动/整合）**: `guide.md` - 存量稿件整合、非线性启动（如先写人设）、进度智能定锚
- **Ideation（创意脑暴）**: `ideation.md` - 创意孵化，生成设定案、金手指、角色表
- **Outline（大纲规划）**: `outline.md` - 规划全书总纲、分卷卷纲、标准化章纲（含场景拆解、伏笔线索）
- **Write（正文写作）**: `write.md` - 逐章正文创作，管理角色状态、伏笔记录、进度追踪
- **Review（质检审稿）**: `review.md` - 扮演主编进行审稿，支持大纲质检、正文质检、设定回溯

### 🔗 与 SOLOENT.md 的集成

本技能包与 **SOLOENT.md**（中央控制台）紧密集成，SOLOENT.md 中各章节通过指针链接到以下文件：

| SOLOENT 章节 | 说明 | 链接文件 |
| :--- | :--- | :--- |
| **§5** | 文风指南 | `1-边界/1.2_文风.md` |
| **§6** | 创作红线（宪法） | `.novelkit/constitution/MASTER.md` |
| **§7.2** | 角色快照（写作前读、写后更新） | `.novelkit/memory/character_state.md` |
| **§7.4** | 伏笔追踪（待收回/已收回/线索链/情感线） | `.novelkit/memory/foreshadowing.md` |
| **§8** | 项目进度与路线图 | 写在 SOLOENT.md 正文 §8，无单独文件 |

各技能会自动更新 SOLOENT.md 的相应章节及上述链接文件，保持项目状态同步。
